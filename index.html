Bubble Sort






  • Array to be sorted in ascending order.
  • Start from first index and move left to right.
  • Compare selected index (i) with next (i+1) index.
  • If value at index (i) is greater than the value at index (i+1) then exchange and increment selected index (i) by 1
  • If value at index (i) is less than or equal to value at index (i+1) simply increment selected index by 1
  • Select the first index 0
